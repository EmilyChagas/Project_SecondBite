## 🚀 Como rodar o projeto

### 📦 Pré-requisitos

* Node.js 18+
* Java 21
* PostgreSQL

---

### ⚙️ Banco de dados

Crie o banco:

```sql
CREATE DATABASE secondbite;
```

---

### 🔐 Variáveis de ambiente

Configure no terminal **antes de rodar o backend**:

```bash
set POSTGRES_USER=postgres
set POSTGRES_PASSWORD=123456
set JWT_SECRET_KEY=meuSegredoUltrapoderosoMuitoLongo1234567890   
```
---

### 🔙 Backend

```bash
mvn spring-boot:run
```

---

### 🔜 Frontend

```bash
npm install
npm run dev
```
